## Separata: Instalación y Configuración Inicial de PostgreSQL

Esta sección cubre el proceso de instalación de PostgreSQL, las herramientas esenciales para interactuar con el gestor de bases de datos, y los primeros pasos, incluyendo los comandos SQL básicos.

### A. Proceso de Instalación

PostgreSQL puede ser instalado en varios sistemas operativos:

*   **Windows:** Se utiliza el instalador oficial.
*   **Linux:** Está disponible para su instalación mediante repositorios o utilizando contenedores como Docker.
*   **Mac (Open System):** La instalación se puede realizar mediante un instalador oficial o a través de *Homebrew* (con el comando `brew postgre`).

El **puerto por defecto** utilizado por PostgreSQL es el **5432**.

La **configuración inicial** después de la instalación incluye:
1. Abrir **PGAdmin 4**.
2. Crear una conexión al servidor.
3. Crear una base de datos.
4. Crear un rol de usuario.
5. Ejecutar consultas utilizando el **Query Tool**.

### B. Herramientas de Interacción

Para la administración y la interacción con PostgreSQL, se utilizan principalmente dos herramientas:

*   **PG Admin (versión 4):** Es la **interfaz gráfica** para administrar PostgreSQL. Es funcional desde el navegador web.
*   **PSQL (PG SQL):** Es la **consola de comandos** que permite interactuar con PostgreSQL.

### C. Primeros Pasos y Comandos Básicos

Una vez configurada la conexión, la interacción con la base de datos se realiza a través de consultas y *scripts*.

*   La ejecución de consultas se lleva a cabo mediante el **Query Tool**.

Los comandos SQL básicos, esenciales para la manipulación de datos, son:
*   **`SELECT`**: Se utiliza para **recuperar datos** desde una o más tablas.
*   **`INSERT`**: Permite **insertar nuevos registros** en una tabla.
*   **`UPDATE`**: Se usa para **modificar datos existentes**.
*   **`DELETE`**: Permite **eliminar registros existentes**.

Además, al iniciar, es posible visualizar las bases de datos predeterminadas mediante el comando `SELECT * FROM pg_database`. Estas bases de datos incluyen:

*   **`postgres`**: La base de datos predeterminada.
*   **`template 1`**: Una base de datos de plantilla.
*   **`template 0`**: La base de datos de plantilla que se puede utilizar por defecto para crear nuevas bases de datos.
